# Portal-Quest
A fun little 2D action adventure game made entiry in Java. Made: 2022
This was made following the tutorial series by RyiSnow: https://www.youtube.com/playlist?list=PL_QPQmz5C6WUF-pOQDsbsKbaBZqXj4qSq

Controls: 
- Move: WASD
- Attack: Enter
- Talk: Enter
- Select: Enter
- Cast: E
- Inventory: C
